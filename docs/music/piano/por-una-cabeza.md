# 一步之遥

老师给了一个简短的改编炫技版，至今未跟上原速（）

说难吧它又很短，虽然左手跳跃性比较大但是肌肉记忆之后这个跨度相对还不那么难找，右手旋律也不算复杂。

但是说简单吧连续12个琶音是真的摸不准，虽然是74的速度但是换算成32分音符确实很快，想提速不出错很难。

很神奇这个曲子感觉越练越烂，现在经常完全找不到音，肌肉记忆都快没了。

因为这个曲子看了《闻香识女人》，很经典的电影，比较推荐。

力度方面老师给的指导会比较多，比如两段琶音从上到下要有力度变化，要么从弱到强，要么从强到弱，但总体不需要太强。下半段的连续八度部分力度在第一个音推到顶点，然后第二个音突然弱下来。以及结尾部分要非常坚定毫无保留推出100%的力量。

艺博馆的闭馆音乐其中有一段就是这首《por-una-cabeza》的小提琴原曲，不得不说这个曲子还是更适合小提琴。
