# 家庭时间（天气之子）

来自一场新海诚音乐会。

难度不高，很容易上手、速度不快且有大片大片的重复。高潮部分蛮好听的。

感觉唯一的不足可能是没有延音踏板听起来会有点单薄，但是如果加上踏板就没有了这种清新感，会有点浑浊。我一般会凭感觉踩三分之一到二分之一的踏板，然后在高潮部分踩到底。
