# 约会（你的名字）

和《家庭时间》同样来自新海诚音乐会。

上手难度也不高，左手力度控不好可能会有点抢戏。

这个好久没弹过了，还是更喜欢《家庭时间》。

弹了这两首曲子之后又补了《你的名字》和《天气之子》的电影，在电影中听到熟悉的曲子，配上应景的画面，一种莫大的享受啊。
