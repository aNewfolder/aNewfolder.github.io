# 伴奏作曲

> 广义的编曲指的是主旋律和伴奏的编写，而狭义的编曲指的是伴奏的编写。编曲，通过织体的加减法营造丰富的声部，通过音符的流动拨动情绪的琴弦。当出现主旋律时，伴奏就是配角，通过和声、副旋律、riff等的协和作用，凸显主旋律的律动，所以伴奏不能抢戏，它就是绿叶。
>
> 同理，在钢琴即兴伴奏/演奏中，钢琴的听感一定是不能掩盖主旋律的，它一般是在比主旋律低八度之外的音区演奏，同时遵循“上密下疏中不空”的原则，越低的音区弹的音要越少，不然低音区的音符会“打架”，听起来很浑浊。
>
> 同时也要注意弹奏力度的强弱变化、延音踏板的应用，让音符更具有情绪变化，在主旋律强时伴奏就弱，在主旋律弱时伴奏就强。
>

## 和弦知识

好吧我承认我的脑海中并没有剩下什么和弦知识，但没关系我可以现学一点）

- 大三和弦：大三度+小三度
- 小三和弦：小三度+大三度
- 减三和弦：小三度+小三度
- 增三和弦：大三度+大三度

那么什么是大三度和小三度呢——

用最笨的方法来说，相邻两个键（无论黑白）之间是1个半音。

- **大三度**：从起始音往上数 **4个半音**
- **小三度**：从起始音往上数 **3个半音**

| 和弦类型          | 构成                       | 例子（以C为根音） |
| ----------------- | -------------------------- | ----------------- |
| **大三和弦** (C)  | 根音 + **大三度** + 纯五度 | C – E – G         |
| **小三和弦** (Cm) | 根音 + **小三度** + 纯五度 | C – E♭ – G        |

大三和弦（如C、G、F）听起来**明亮、坚定**；小三和弦（如Cm、Am、Em）听起来**忧伤、柔和**。这种色彩差异完全来自于中间那个音是大三度还是小三度。



根据我的感受考虑大三度和小三度会非常麻烦，所以更方便的方法是直接查表：

| 大三和弦    | 小三和弦    |
| :---------- | :---------- |
| C (C–E–G)   | Cm (C–E♭–G) |
| D (D–F♯–A)  | Dm (D–F–A)  |
| E (E–G♯–B)  | Em (E–G–B)  |
| F (F–A–C)   | Fm (F–A♭–C) |
| G (G–B–D)   | Gm (G–B♭–D) |
| A (A–C♯–E)  | Am (A–C–E)  |
| B (B–D♯–F♯) | Bm (B–D–F♯) |

从表里也能看出来我最喜欢的也是最常见的就是C、Dm、Em、F、G、Am，因为这些没有黑键，非常好上手...



同时和弦也可以转位，构成转位三和弦：

C-E-G --> E-G-C

那么转位的意义又在哪里呢，个人最直观的感受是可以让演奏集中在键盘的一小块区域内，降低左手去找音的难度（）

同时也有助于让旋律线条更流畅，不会有很强的跳跃感。

## 织体

一般我们最常用的是主调织体，也就是一条主旋律+和弦伴奏。可以是左手和弦+右手主旋律或左右手都弹和弦，绝大多数流行歌与钢伴都会采用这种形式。

织体的英文是texture，其重点就在于如何把和弦变换多种形式去表现，这也是即兴演奏/伴奏、歌曲伴奏的根本。具体来说，我们拿到一个歌曲的吉他谱（如图）：

![](https://kevin13579me-1422109417.cos.ap-shanghai.myqcloud.com/img/20260722225350625.jpg)

上面会用简谱标出主旋律，同时再标上对应的和弦，那么作为钢琴来讲我们就要把一个简单的C弹得丰富、悦耳。

水平较高的演奏者（比如我的老师）可以以非常华丽的方式去演奏一个和弦，因此他们的即兴演奏在普通人听来就是难度非常高的乐曲。不过作为初学者而言，还是先从最基础的几种织体开始：

- 柱式和弦：同时演奏和弦中的所有音。
- 分解和弦：先后弹奏和弦中的音，比如对于C和弦，演奏为C-E-G-E。著名的《菊次郎的夏天》基本就是用这种方式组织的。
- 半分解和弦：单独弹奏最低音，同时弹奏其余音。（这个我没太用过）

下面是一些让织体更复杂的方法：

- 添加节奏：切分、附点、三连音等，或在固定的拍上演奏，比如把4/4拍的一个小节均分成8个部分，然后弹奏1&4、1&4&7、1&4&7&8、1&3&4&5&6&7&8等等
- 改变音区：标准钢琴有88个键，整个钢琴都属于你
- 使用琶音：这个很考验手速
- ...

方法有很多，勤加练习会越来越熟练的。

老师当时布置的任务是给蔡健雅的《被驯服的象》和弦伴奏，整首曲子（除了间奏部分）基本都可以用Am-G-F-E循环去写，间奏部分当时老师引导我写了一点爵士风格。感兴趣可以试试。
