# 乒乓球

高考后开始培养的爱好。简单写一些需要注意的点。

## 接球

- 正手：蹬腿转腰，用身体的力量，用脚找球
- 反手：以胳膊肘为支点，小臂向外打，不要敲，尽可能拍头朝左
- 正手短：上右腿，球拍贴着台面直线伸出去，向上收，打出力量
- 反手短：上右腿，先俯身拍头朝左等球，要有等的动作
- 侧身：侧过来之后直接把重心放在右腿上，等球，蹬腿转腰
- 反手出台：垫步整个人到左边，同反手短
- 扣杀：右手往后上方等球，垫步，在上方往里收
- 反手撕：在反手动作基础上准备时拍头朝下，球拍前部触球，手腕的力量
- 反手搓球：低高低（胳膊肘放低，手腕抬高，拍头放低），先上右脚俯身等球，拍子放到球前面，然后再用身体的力量向前向下
- 正手搓球：胳膊肘夹紧身体，垫步左腿右腿一起上，俯身，用身体的力量，接触球的瞬间向前向下，不需要刻意砍
- 正手拉下旋：极致的左前右后，重心放在右腿，左肩转过去向右，大臂放松，手臂自然下垂，球拍放低，外侧向上向前画圆收回来，拍头向下，手腕内钩往对方正手位打，蹬腿发力，把身体转正
- 正手挑打：针对下旋球，较高/较长直接攻球姿势瞬间向前向上发力，注意加上向上，较短可以先搓球姿势等球，触球瞬间迅速手腕内翻
- 反手拧拉：尽可能让身体与球接近平行，球越靠左越好接，胳膊肘架高撑住，只动小臂，从拍头朝下向外打，小臂要出去，先上右脚俯身在球前面等球，必须有等球的动作，弹起来较低时直接拧出去，不要等球弹高

## 接发球

- 拍头朝前，俯身向前趴，观察对方出手时拍面与球的接触情况

## 发球

- 正手发下旋：腰基本抵在球台左边角，左前右后，中指抵在胶皮与底板交界处，拇指食指握拍，要有手腕动作。俯身趴低一点，左手略向前放，左右左重心转换，上抛时两手同时向上，向后引手，等球落低再击球，用拍的前部触球，往腰间砍，把球打出去之后再起身，上左腿恢复正常站位，发球要果断，然后迅速把手型还原，注意步伐移动不要站死
- 正手发上旋：前置动作与下旋完全一致，等球落低触球前的一刹那把拍面翻过来发急奔长球，冲着两个角和底线发，重点在于打对手一个措手不及
- 还没学过反手
