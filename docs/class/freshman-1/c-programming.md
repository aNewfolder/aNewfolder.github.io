## 课程总结与建议 & 教师评价

作为编程零基础的小白，lz开学前还是对计算机有些畏惧的。从学期初到学期末，难度呈现明显的递增趋势，从前几章的并不费力，到第六章数组开始略有难度，第七章指针开始听不懂，后面渐渐完全放弃了听课，转而试图通过豆包、PTA上的习题和小测前两天的补天进行学习，最终还是败在今年难出天际的期末考试上。

学习方法上，由于C程有许多细碎且易错的知识点，强烈建议从学期初就养成整理笔记的习惯，无论是纸质还是电子。lz选择了非常普通的word文档，通过截图、复制粘贴、手敲等方式收集起了上课PPT重点（部分，因为后面的课大多没有认真听），PTA题目、豆包讲解与解析等内容，至少在无论是小测还是最终的期末复习时有材料可看。就今年的期末考试来看后半部分的指针、链表、文件等难度较大的部分是主要考察点，无论是题量还是计算量上都比平常的练习或小测要高，也因此导致取得高分并不容易。

个人认为刘新国老师还是相对比较不错的，上课会很认真地解释一些概念与例题努力让同学们理解（效果另论），无论是习题课还是钉钉答疑群都会亲自答疑，部分同学反馈脾气暴躁的问题我并没太感受到（也可能是高中已经见多了暴躁的老师qwq），但那种空有一腔热情却无人响应的失落感的确存在，反过来说也是因为刘老师非常认真负责。

但课堂效果似乎不甚如意，一方面是lz确实没有及时预习复习导致大量内容听不懂，另一方面东1A昏暗而拥挤的教室并不构成一个很好的学习环境，自学或听智云同样是一种选择。

虽然C语言在实际编程中似乎应用范围并不太宽，但对于lz这样没有计算机基础的小白来讲还是可以对编程语言的基本逻辑建立一个很好的框架，后期再接触python等也会相对容易很多。

以及，本课程的别名是"C尖"。
