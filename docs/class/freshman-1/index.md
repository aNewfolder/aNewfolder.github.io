## 写在前面

关于要不要放上绩点犹豫了很久，但出于自己刚开学时浏览类似帖子还是非常希望看到分享者的绩点作为一种参照，问了身边的朋友同样如此，还是最终加上了课程得分，毕竟大部分预置课的目标群体是未来的小朋友而非早已上完这些课程的大朋友们（没有任何褒贬之意），而附有得分的选修课程可能也更有助于读者了解老师的给分情况。

而小朋友如果看到很多分数偏高的分享也不要产生过高的心理压力和预期。一方面幸存者偏差是必然存在的，愿意分享的同学大多至少获得了相对满意的成绩，浏览这些也未免不是一种正向激励，但同时也请记住，一门课的绩点成因是多元的，中学的基础、个人兴趣、性格因素、思维方式等等都可能影响最终的成绩，一时没能获得所谓的高分完全不代表自己低人一等或是笨这样子。同时，绩点作为一种单一评价体系本身也是缺陷重重的，更重要的是想清楚自己究竟想要什么，未来想做什么，自己在未来更需要哪些技能和品质。就像我们每个人为之奋斗三年甚至更多的高考成绩在进入大学后几乎变得一文不值，绩点本身也只是在有限的场景下能体现实用性，本科毕业步入人生下一阶段它同样会迅速贬值。我一直觉得区分努力和恶性竞争的重要特征就是自己是否在享受做一些事情的过程，在感兴趣的任务上花费更多精力算不上恶竞，强迫自己单纯为了分数去做令自己厌恶的事情则并不太值得推崇。

当然，作为自身同样刚刚步入大学半年的我，视角必定还是狭隘和不准确的。话说回来，所有经验分享的帖子毕竟是别人的经验，自己的路依旧要靠自己一步步去走。不过，如果能帮助后来的人少走一点弯路，避开一些本不必跳入的坑，我想就是一代代前辈们敲下这些文字的很大一部分心愿了。

——来自大一寒假

P.S. 介于自己大部分参考资料来自CC98，直接放上怕有剽窃之嫌（好吧主要是有点嫌麻烦）...但是！随着大语言模型的发展我们会发现参考资料的制作将会越来越简单，所以最终回到善用工具+善用搜索。大二上开始会尽量产出电子笔记持续更新的（*希望如此）
