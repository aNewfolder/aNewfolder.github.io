# 《人类群星闪耀时》

> 斯蒂芬·茨威格

- （穆罕默德）这个令人瞠目结舌的大胆想法，即让几百艘战船越过多山的岬角地带，首先就显得十分荒诞不经，简直难以实现，所以拜占庭人和加拉塔的热那亚人几乎从未把其纳入自己的战略计划加以斟酌，就好像之前的罗马人以及之后的奥地利人同样没有考虑到汉尼拔和拿破仑的军队会如此神速地越过阿尔卑斯山一样。

  这种天才往往嘲讽那些按照战争规则所进行的战争，在特定的时刻并不因循守旧，而是随机应变、即兴创造。

- 一些极其珍贵的绘画作品被销毁，那些最最精美杰出的雕塑作品则被榔头砸毁，一些凝聚了几个世纪智慧的、保存了希腊人思想和诗歌的不朽财富——书籍被焚毁或者被不经意地扔掉了。人类将永远无法完整地知晓整个事件的全部真相，在那个命运攸关的关键时刻，这扇敞开的凯尔卡城门究竟带来了什么样的灾难，在洗劫罗马、亚历山大城以及拜占庭时，人类的精神世界究竟遭受了多大的损失。

- 倘若格鲁希这时候能够鼓起勇气，大胆果断地相信自己、相信这种显而易见的信号，而不是盲目地服从命令，那么法兰西就得救了。但是这个毫无主见的家伙始终只会听命于条条框框的规定，而从来也不会听从命运的召唤。

  这就是决定滑铁卢命运的铁骰子。

  公民的所有美德，小心、顺从、勤勉和谨慎，也许是对付平淡日子各种要求的利器，但是在决定命运伟大瞬间的热情里，它们全都被化为乌有。命运，这个尘世间的另一位神灵，它只愿意用热情的双臂把勇敢者高高举起，送入英雄们的天堂。

- 他听见了弱小者的声音；始终遭受欺凌者那阴郁的哀怨声；从未展露笑容的孤寂者之哀鸣声；孩子们的抽啜声、哭闹声；那些被遗弃者、麻木不仁者和受到侮辱者发出的声音；那些籍籍无名的殉道者发出的呼唤声。

- 这一点不用担心我们，万事皆有可能，只有这一点不会。

- 斯科特：他们于是作出决定，不再迈步迎向灾难，而是自豪地忍受死亡以及其他的不幸降临。他们缓缓地爬进各自的睡袋，却始终都没有就自己最后遭受的痛楚向这个世界发出过一声叹息。
