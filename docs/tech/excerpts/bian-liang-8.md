# 《变量8》

“00 后”在历史上将有什么样的地位呢？这一代人不如“60后”和“70后”野心勃勃、敢作敢为，也不如“80后”和“90后”兢兢业业、有规则感。他们对权威没有敬畏感，对表达有极度的渴求，可又不知道说什么才好。他们不关心历史，也不关心时事，胸怀不够宽广，想象力也不够壮阔。他们更关心自我，对捍卫自己极度敏感，一批评就炸毛。他们宁可选择不升职、不加班、不买房、不结婚、不生娃、不恋爱。他们注定不是光芒万丈的一代，但他们可能是最后一代仍然受到重视的年轻人。别忘了，随着人口老龄化，年轻人而不是老年人将成为社会边缘群体。以后，年轻人想什么，只怕越来越少的人愿意知道。这是我们需要关注他们的最重要的原因。但我们怎样才能和他们沟通交流呢？说教已然没用，年轻人对上一代的建议就是：不要给我建议。启蒙毫无价值，上一代人关心的，年轻人通通无感。甚至威胁都威胁不了他们，只要他们选择了躺平，很多威胁便立刻变得毫无用武之地。美是唯一可以激发他们兴趣的话题。哪里有美，哪里就有年轻人。
