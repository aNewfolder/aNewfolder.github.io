# 《夜色温柔》

> 菲茨杰拉德

- 这段话说的是第一次世界大战中最残酷的凡尔登战役。这场战役号称是"凡尔登绞肉机"。十个月的战斗，双方伤亡100多万人。太惨烈了。

  菲茨杰拉德说，这场战役，"绝不可能重来一遍了，很长时间内都不可能。这种战争需要信仰，需要多年积攒下来的富足和安全感，还需要各阶级之间确定而明晰的关系。你必须有比你所能记忆的还要悠久而且使你能全心全意的情感根源。你必须记得皇太子和他的未婚妻的明信片，还有瓦朗斯的小咖啡馆、柏林菩提树下大街上的啤酒屋以及市政厅的婚礼，记得去德比看赛马会，记得爷爷唇边的胡子……这是一场爱的战役——消耗掉了一个世纪的中产阶级的爱……我整个美丽且安全的世界，都随着这一阵爆炸带过的狂风，在我面前炸得粉碎，化为乌有了。"

- 你会感觉到自己的影子在那些注视着你的人的眼前丝丝滑过。你不再与世隔绝，但我猜想，你必须接触生活，才能从生活中跳出来。

- 美国内战后，一些南方年轻人来到北方，他们注重举止，努力追求优雅。

  狄克从父亲那里学到了这些优点。他经常一边努力做到优雅得体，一边鄙视这种做法，认为这样反对的不是丑陋的自私自利，而是自私自利的丑陋的外表。

- 彬彬有礼就意味着大家都很脆弱，所以需要谨慎地对待。
